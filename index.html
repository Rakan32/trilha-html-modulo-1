Primeiros computadores

A primeira geração começou com Mark 1 uma maquina de calculos eletromecânica que utilizava válvulas no manuseio, outro da geração seguiam essa mesma lógica,
Já na segunda geração foi um marco pois ao invés de usarem válvulas agora usavam trasistores no seu componente, assim tornando mais pequeno para época,
A terceira geração vem acompanhada dos circuitos integrados dimuniundo mais ainda o tamanho dos computadores,
já a quarta geração é a atual que vivemos mudando de circuitos integrados para micro processadores surge também o Personal computer (pc).

Client

Cliente é o dispositivo que consome uma determinada informação de uma rede de internet, ou seja Notebooks, celulares, computadores.
resumindo o princípio de cliente é consumir informações.

Servers

É o componente que oferece serviços ou alguns recursos ao cliente, esses servidores podem ser tanto um computador projetado para gerenciar ou fornecer serviços,
quanto um software executado em um servidor de rede

Estrutura básica HTML

O documento sempre iniciam pelo que se chama de Estrutura básica, você sempre terá que começar seu HTML por esse código boa parte dos editores como Vscode já são compostos de atalhos para iniciar HTMLs
com essa estrutura e torna fácil compreender com a divisão de blocos das tags essenciais que são, Definição do documento (doctype), Cabeça (head), Corpo (body).
